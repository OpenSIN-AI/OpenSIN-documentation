# Code of Conduct

OpenSIN Documentation is a shared collaboration space for maintainers, contributors, and agents.

## Expected behavior

- Be respectful and precise.
- Keep feedback evidence-based and actionable.
- Separate facts, assumptions, and speculation.
- Protect private information and internal-only links.

## Unacceptable behavior

- Harassment, threats, or insults
- Doxxing or publishing private information
- Secret leakage or screenshots that expose credentials
- Spamming, trolling, or intentionally misleading claims

## Enforcement

If behavior needs escalation, use the repository's private or maintainer-only channels when available. For sensitive issues, prefer the private security path instead of a public issue.
