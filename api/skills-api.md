# Skills API API

> **Version:** v1 | **Status:** ✅ Active

## Overview

Skills API — Skill management.

## Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/v1/skills` | GET | List all |
| `/api/v1/skills/{id}` | GET | Get by ID |
| `/api/v1/skills` | POST | Create new |
| `/api/v1/skills/{id}` | PUT | Update |
| `/api/v1/skills/{id}` | DELETE | Delete |

## Authentication

All endpoints require Bearer token authentication.

## Rate Limits

- 100 requests per minute
- 10,000 requests per hour
