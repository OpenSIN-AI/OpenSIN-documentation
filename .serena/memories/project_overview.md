OpenSIN-documentation is the OpenSIN documentation site for the OpenSIN-AI ecosystem. It is primarily a VitePress documentation project with docs under docs/, public assets under public/, VitePress config under .vitepress/, and a test suite under tests/. The current task also includes a standalone Python worker file at repo root (heypiggy_vision_worker.py) used for control-plane/vision-gated automation work related to HeyPiggy safeguards.
