## Quick Start

```bash
# Clone the repository
git clone https://github.com/OpenSIN-AI/OpenSIN.git
cd OpenSIN

# Install and start
pip install opensin-cli
opensin agent create researcher --model gpt-4
opensin agent test researcher --prompt "What is AI?"
```
