# Agent Creation Example

> **Category:** A2A | **Difficulty:** Intermediate

## Overview

Example of creating a new A2A agent for OpenSIN-AI.

## Steps

1. Use `create-a2a-sin-coder` skill
2. Define agent card
3. Configure MCPs
4. Deploy to HF Space
5. Register in fleet

## Agent Card

```json
{
  "name": "sin-example",
  "description": "Example agent",
  "version": "1.0.0",
  "capabilities": ["chat", "task"]
}
```
