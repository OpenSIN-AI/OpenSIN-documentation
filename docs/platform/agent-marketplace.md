# Agent Marketplace

> **Status:** 🚧 In Development | **Type:** Platform Feature

## Overview

Marketplace for discovering and installing OpenSIN agents.

## Features

- Browse available agents
- View agent cards
- Install with one click
- Rate and review agents

## Agent Card Schema

```json
{
  "name": "agent-name",
  "version": "1.0.0",
  "description": "Agent description",
  "capabilities": ["chat", "task"],
  "mcpDependencies": ["mcp-1", "mcp-2"]
}
```
