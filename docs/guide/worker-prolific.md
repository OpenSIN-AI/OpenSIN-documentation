# Prolific Worker

The Prolific worker is a specialist automation surface inside the wider OpenSIN
fleet. It is not a primary end-user product surface and should be read as a
specialized operations capability, not as the main entry point to OpenSIN.

## What It Is

- browser-driven task automation for Prolific workflows
- human-like interaction safeguards
- specialist runtime for survey-oriented execution

## Scope Note

This public page intentionally avoids exposing internal-only runtime details,
host-specific secrets, or brittle implementation assumptions. For canonical
ownership and current runtime placement, verify the owning repo and fleet
registry before extending this surface.
