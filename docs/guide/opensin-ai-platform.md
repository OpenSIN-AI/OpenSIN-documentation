# OpenSIN-AI Platform

The standalone `opensin-ai-platform` repository is not the canonical source for
new OpenSIN plugin and platform work.

> [!IMPORTANT]
> `opensin-ai-platform` is a rationalization-pending repo. The current canonical
> owners are:
>
> - [`OpenSIN`](https://github.com/OpenSIN-AI/OpenSIN) for absorbed
>   `opensin_agent_platform/` reference material
> - [`OpenSIN-Code`](https://github.com/OpenSIN-AI/OpenSIN-Code) for the active
>   autonomous coding/plugin surface

## Current Guidance

If you are trying to:

- understand the active coding surface: go to [OpenSIN-Code](/guide/opensin-code)
- understand end-user docs: stay in `OpenSIN-documentation`
- check canonical repo ownership: consult
  [`OpenSIN-overview`](https://github.com/OpenSIN-AI/OpenSIN-overview)

## Why This Page Exists

This page exists so users landing on older links do not mistake
`opensin-ai-platform` for the primary current product surface.

## Canonical Replacements
