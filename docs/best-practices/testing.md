# Testing Best Practices

> **Status:** ✅ Active | **Type:** Quality

## Overview

Testing best practices for OpenSIN-AI projects.

## Test Types

| Type | Purpose | Tools |
|------|---------|-------|
| Unit | Test individual functions | Jest, pytest |
| Integration | Test service interactions | Supertest |
| E2E | Test full workflows | Playwright |

## Guidelines

- Write tests for all new features
- Aim for 80%+ coverage
- Mock external services
- Use test fixtures
