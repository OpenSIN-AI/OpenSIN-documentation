# 👥 Community Onboarding Guide

Willkommen bei OpenSIN! Wir sind die weltweit erste AI Agent Organization – 100+ spezialisierte Agenten arbeiten zusammen.

## Dein erster Beitrag

### Good First Issues

Suche nach Issues mit Label `good-first-issue`:

1. Neues Tool zu opensin-tool-system hinzufügen
2. Dokumentation für ein bestehendes Package schreiben
3. Bug in einem Agent runtime.ts fixen
4. Tests für ungetesteten Code schreiben

### Contributor Tiers

| Tier | Voraussetzungen | Benefits |
|------|----------------|----------|
| **Newcomer** | Erster PR gemerged | Discord Rolle, Swag |
| **Contributor** | 5 PRs gemerged | Early Access auf Features |
| **Maintainer** | 20 PRs + Reviews | Write Access |
| **Core Team** | Architektur-Entscheidungen | Stimmrecht |

## Schnellstart

```bash
# 1. Repo klonen
git clone https://github.com/OpenSIN-AI/OpenSIN.git

# 2. Dependencies installieren
cd OpenSIN && npm install

# 3. Build
npm run build

# 4. Tests ausführen
npm test

# 5. Ersten PR erstellen
git checkout -b docs/my-update
# ... Änderungen machen ...
git commit -m "docs: add XYZ"
git push origin my-update
```
