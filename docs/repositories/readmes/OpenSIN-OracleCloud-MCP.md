# OpenSIN-OracleCloud-MCP

> **Organization:** OpenSIN-AI | **Status:** Active | **Visibility:** Private

## Overview

OpenSIN-OracleCloud-MCP is a core repository within the OpenSIN-AI ecosystem.

## Tech Stack

| Component | Technology |
|-----------|------------|
| **Language** | TypeScript / Python |
| **Framework** | Node.js / FastAPI |
| **Database** | PostgreSQL (Supabase) |
| **Deployment** | Docker / HF Spaces |
| **CI/CD** | n8n Workflows |

## Quick Start

```bash
git clone git@github.com:OpenSIN-AI/OpenSIN-OracleCloud-MCP.git
cd OpenSIN-OracleCloud-MCP
npm install  # or pip install -r requirements.txt
npm run dev
```
