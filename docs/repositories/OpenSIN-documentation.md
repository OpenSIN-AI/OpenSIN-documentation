# OpenSIN-documentation

## Overview

Core repository for OpenSIN-documentation within the OpenSIN-AI ecosystem.

## Tech Stack

- **Language:** TypeScript / Python
- **Framework:** Node.js / FastAPI
- **Database:** PostgreSQL (Supabase)
- **Deployment:** Docker / HF Spaces

```

├── src/                 # Source code
├── tests/               # Test suites
```

## Getting Started

```bash
git clone git@github.com:OpenSIN-AI/OpenSIN-documentation.git
cd OpenSIN-documentation
npm install
npm run dev
```
