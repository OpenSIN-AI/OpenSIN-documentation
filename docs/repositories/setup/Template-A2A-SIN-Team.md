# Template-A2A-SIN-Team — Setup Guide

> **Repository:** Template-A2A-SIN-Team | **Category:** Setup

## Prerequisites

- Node.js 18+
- Python 3.11+
- Git
- OpenCode CLI

## Installation

```bash
git clone git@github.com:OpenSIN-AI/Template-A2A-SIN-Team.git
cd Template-A2A-SIN-Team
npm install  # or pip install -r requirements.txt
```

## Configuration

```bash
cp .env.example .env
# Edit .env with your credentials
```

## Running

```bash
npm run dev  # or python -m template_a2a_sin_team
```

## Testing

```bash
npm test  # or pytest
```

## Deployment

```bash
npm run build
npm run deploy
```
