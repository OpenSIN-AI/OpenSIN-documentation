# A2A-SIN-Code-GitLab-LogsCenter — Setup Guide

> **Repository:** A2A-SIN-Code-GitLab-LogsCenter | **Category:** Setup

## Prerequisites

- Node.js 18+
- Python 3.11+
- Git
- OpenCode CLI

## Installation

```bash
git clone git@github.com:OpenSIN-AI/A2A-SIN-Code-GitLab-LogsCenter.git
cd A2A-SIN-Code-GitLab-LogsCenter
npm install  # or pip install -r requirements.txt
```

## Configuration

```bash
cp .env.example .env
# Edit .env with your credentials
```

## Running

```bash
npm run dev  # or python -m a2a_sin_code_gitlab_logscenter
```

## Testing

```bash
npm test  # or pytest
```

## Deployment

```bash
npm run build
npm run deploy
```
