# OpenSIN-Apple-FaceTime — Setup Guide

> **Repository:** OpenSIN-Apple-FaceTime | **Category:** Setup

## Prerequisites

- Node.js 18+
- Python 3.11+
- Git
- OpenCode CLI

## Installation

```bash
git clone git@github.com:OpenSIN-AI/OpenSIN-Apple-FaceTime.git
cd OpenSIN-Apple-FaceTime
npm install  # or pip install -r requirements.txt
```

## Configuration

```bash
cp .env.example .env
# Edit .env with your credentials
```

## Running

```bash
npm run dev  # or python -m opensin_apple_facetime
```

## Testing

```bash
npm test  # or pytest
```

## Deployment

```bash
npm run build
npm run deploy
```
