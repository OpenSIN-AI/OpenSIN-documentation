# Community Guidelines

## Welcome!

Welcome to the OpenSIN community! We're excited to have you here.

## Code of Conduct

### Our Pledge

We pledge to make participation in our community a harassment-free experience for everyone.

### Our Standards

**Positive behavior:**
- Being respectful and inclusive
- Being welcoming to newcomers
- Being constructive in feedback
- Being collaborative in problem-solving

**Unacceptable behavior:**
- Harassment or discrimination
- Personal attacks or trolling
- Publishing private information
- Other unprofessional conduct

### Reporting

Report violations to [conduct@opensin.ai](mailto:conduct@opensin.ai).

## Contributing

### Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write tests
5. Submit a pull request

### Pull Request Guidelines

- Keep PRs focused and small
- Include tests for new features
- Update documentation
- Follow existing code style

### Code Review

- Be constructive and respectful
- Focus on the code, not the person
- Explain your reasoning
- Suggest alternatives when possible

We recognize contributors who:
- Submit high-quality PRs
- Help others in discussions
- Improve documentation
- Report bugs
- Share best practices

Thank you for being part of the OpenSIN community!
