# Community Guidelines

## Welcome!

Welcome to the OpenSIN community!

## Code of Conduct

### Our Pledge

We pledge to make participation a harassment-free experience.

### Our Standards

**Positive:** Respectful, welcoming, constructive, collaborative.

**Unacceptable:** Harassment, personal attacks, publishing private info.

### Reporting

Report violations to [conduct@opensin.ai](mailto:conduct@opensin.ai).

## Contributing

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write tests
5. Submit a PR
