# Getting Started with OpenSIN-AI

> **Status:** ✅ Active | **Type:** Onboarding

## Overview

Quick start guide for new OpenSIN-AI contributors.

## Prerequisites

- Node.js 18+
- Python 3.11+
- Git
- OpenCode CLI

## Setup

```bash
# Clone repository
git clone git@github.com:OpenSIN-AI/OpenSIN-backend.git
cd OpenSIN-backend

# Install dependencies
npm install

# Start development
npm run dev
```

## First Contribution

1. Find a "good first issue"
2. Comment on the issue
3. Create a branch
4. Make your changes
5. Submit a PR
