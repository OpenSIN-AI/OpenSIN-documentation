# API Reference

OpenSIN V1 exposes three API surfaces that matter publicly:

1. [API Overview](/api/overview)
2. [OpenSIN Backend V1 API](/api/backend-v1)
3. [A2A Protocol API](/api/a2a)
4. [Team Manifest Contract](/api/team)

If you are integrating with `chat.opensin.ai`, start with the backend control-plane page.
If you are integrating agents with each other, start with the A2A page.
If you are building marketplace or bundle logic, start with the team manifest contract.
