# Supabase Setup Guide

> **Status:** ✅ Active | **Type:** Database Setup

## Overview

Supabase provides PostgreSQL database, authentication, and real-time subscriptions for OpenSIN.

## Tables

| Table | Purpose |
|-------|---------|
| `antigravity_tokens` | Token pool management |
| `openai_tokens` | OpenAI token pool |
| `agent_registry` | Agent registration |
| `work_items` | Task queue |
| `incidents` | Incident tracking |
