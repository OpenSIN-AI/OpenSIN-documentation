# Sin Tiktok Shop — Troubleshooting Guide

> **MCP:** sin-tiktok-shop | **Category:** Troubleshooting | **Status:** Active

## Common Issues

### Issue 1: MCP Server Not Starting

**Symptoms:**
- MCP server fails to start
- Connection refused errors
- No tools available

**Resolution:**
1. Check MCP installation
2. Verify command path in configuration
3. Check environment variables
4. Review MCP logs

### Issue 2: Tool Call Failed

**Symptoms:**
- Tool returns error response
- Unexpected behavior
- Timeout errors

**Resolution:**
1. Verify tool parameters
2. Check API credentials
3. Review tool documentation
4. Test with minimal parameters

### Issue 3: Authentication Failed

**Symptoms:**
- 401 errors from tool calls
- Authentication error messages
- Tools return empty results

**Resolution:**
1. Verify API key is set
2. Check API key has not expired
3. Update credentials in configuration
4. Test API key manually

## Diagnostic Commands

Check MCP server status, test tool calls, and view MCP logs.
