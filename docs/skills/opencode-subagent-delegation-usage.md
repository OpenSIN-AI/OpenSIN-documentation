# Opencode Subagent Delegation — Usage Guide

> **Skill:** opencode-subagent-delegation | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Opencode Subagent Delegation** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about opencode subagent delegation.

## Manual Activation

```bash
opencode run "use skill opencode-subagent-delegation to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use opencode-subagent-delegation
```

### Example 2: Advanced Usage
```
/use opencode-subagent-delegation with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/opencode-subagent-delegation/`
2. Verify the SKILL.md exists
3. Restart OpenCode
