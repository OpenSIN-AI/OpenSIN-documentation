# Sovereign Research — Usage Guide

> **Skill:** sovereign-research | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Sovereign Research** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about sovereign research.

## Manual Activation

```bash
opencode run "use skill sovereign-research to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use sovereign-research
```

### Example 2: Advanced Usage
```
/use sovereign-research with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/sovereign-research/`
2. Verify the SKILL.md exists
3. Restart OpenCode
