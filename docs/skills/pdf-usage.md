# Pdf — Usage Guide

> **Skill:** pdf | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Pdf** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about pdf.

## Manual Activation

```bash
opencode run "use skill pdf to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use pdf
```

### Example 2: Advanced Usage
```
/use pdf with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/pdf/`
2. Verify the SKILL.md exists
3. Restart OpenCode
