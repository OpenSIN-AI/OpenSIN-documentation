# Cloudflare Deploy — Usage Guide

> **Skill:** cloudflare-deploy | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Cloudflare Deploy** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about cloudflare deploy.

## Manual Activation

```bash
opencode run "use skill cloudflare-deploy to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use cloudflare-deploy
```

### Example 2: Advanced Usage
```
/use cloudflare-deploy with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/cloudflare-deploy/`
2. Verify the SKILL.md exists
3. Restart OpenCode
