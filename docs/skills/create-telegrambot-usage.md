# Create Telegrambot — Usage Guide

> **Skill:** create-telegrambot | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Create Telegrambot** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about create telegrambot.

## Manual Activation

```bash
opencode run "use skill create-telegrambot to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use create-telegrambot
```

### Example 2: Advanced Usage
```
/use create-telegrambot with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/create-telegrambot/`
2. Verify the SKILL.md exists
3. Restart OpenCode
