# Enterprise Deep Debug — Usage Guide

> **Skill:** enterprise-deep-debug | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Enterprise Deep Debug** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about enterprise deep debug.

## Manual Activation

```bash
opencode run "use skill enterprise-deep-debug to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use enterprise-deep-debug
```

### Example 2: Advanced Usage
```
/use enterprise-deep-debug with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/enterprise-deep-debug/`
2. Verify the SKILL.md exists
3. Restart OpenCode
