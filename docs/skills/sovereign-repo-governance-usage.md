# Sovereign Repo Governance — Usage Guide

> **Skill:** sovereign-repo-governance | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Sovereign Repo Governance** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about sovereign repo governance.

## Manual Activation

```bash
opencode run "use skill sovereign-repo-governance to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use sovereign-repo-governance
```

### Example 2: Advanced Usage
```
/use sovereign-repo-governance with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/sovereign-repo-governance/`
2. Verify the SKILL.md exists
3. Restart OpenCode
