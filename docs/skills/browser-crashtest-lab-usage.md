# Browser Crashtest Lab — Usage Guide

> **Skill:** browser-crashtest-lab | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Browser Crashtest Lab** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about browser crashtest lab.

## Manual Activation

```bash
opencode run "use skill browser-crashtest-lab to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use browser-crashtest-lab
```

### Example 2: Advanced Usage
```
/use browser-crashtest-lab with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/browser-crashtest-lab/`
2. Verify the SKILL.md exists
3. Restart OpenCode
