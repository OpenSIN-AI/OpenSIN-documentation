# Create Github Account — Usage Guide

> **Skill:** create-github-account | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Create Github Account** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about create github account.

## Manual Activation

```bash
opencode run "use skill create-github-account to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use create-github-account
```

### Example 2: Advanced Usage
```
/use create-github-account with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/create-github-account/`
2. Verify the SKILL.md exists
3. Restart OpenCode
