# Nvidia 3D Forge — Usage Guide

> **Skill:** nvidia-3d-forge | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Nvidia 3D Forge** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about nvidia 3d forge.

## Manual Activation

```bash
opencode run "use skill nvidia-3d-forge to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use nvidia-3d-forge
```

### Example 2: Advanced Usage
```
/use nvidia-3d-forge with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/nvidia-3d-forge/`
2. Verify the SKILL.md exists
3. Restart OpenCode
