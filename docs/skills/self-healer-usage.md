# Self Healer — Usage Guide

> **Skill:** self-healer | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Self Healer** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about self healer.

## Manual Activation

```bash
opencode run "use skill self-healer to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use self-healer
```

### Example 2: Advanced Usage
```
/use self-healer with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/self-healer/`
2. Verify the SKILL.md exists
3. Restart OpenCode
