> **Skill:** create-a2a-mcp | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Create A2A Mcp** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about create a2a mcp.

## Manual Activation

```bash
opencode run "use skill create-a2a-mcp to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use create-a2a-mcp
```

### Example 2: Advanced Usage
```
/use create-a2a-mcp with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/create-a2a-mcp/`
2. Verify the SKILL.md exists
3. Restart OpenCode
