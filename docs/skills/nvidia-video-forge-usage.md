# Nvidia Video Forge — Usage Guide

> **Skill:** nvidia-video-forge | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Nvidia Video Forge** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about nvidia video forge.

## Manual Activation

```bash
opencode run "use skill nvidia-video-forge to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use nvidia-video-forge
```

### Example 2: Advanced Usage
```
/use nvidia-video-forge with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/nvidia-video-forge/`
2. Verify the SKILL.md exists
3. Restart OpenCode
