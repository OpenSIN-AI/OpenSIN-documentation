# Check Plan Done — Usage Guide

> **Skill:** check-plan-done | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Check Plan Done** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about check plan done.

## Manual Activation

```bash
opencode run "use skill check-plan-done to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use check-plan-done
```

### Example 2: Advanced Usage
```
/use check-plan-done with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/check-plan-done/`
2. Verify the SKILL.md exists
3. Restart OpenCode
