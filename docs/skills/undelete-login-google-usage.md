# Undelete Login Google — Usage Guide

## Overview

This guide provides detailed usage instructions for the **Undelete Login Google** skill in OpenCode.

The skill is automatically triggered when the user asks about undelete login google.

## Manual Activation

```bash
opencode run "use skill undelete-login-google to [task]"
```

## Examples


## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/undelete-login-google/`
2. Verify the SKILL.md exists
3. Restart OpenCode
