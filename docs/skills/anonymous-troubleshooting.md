# Anonymous — Troubleshooting Guide

> **Skill:** anonymous | **Category:** Troubleshooting | **Status:** Active

## Common Issues

### Issue 1: Skill Not Loading

**Symptoms:**
- Skill does not appear in skill list
- Manual activation fails
- No skill-related actions available

**Resolution:**
1. Check skill is installed
2. Verify SKILL.md exists and is valid
3. Restart OpenCode
4. Check OpenCode logs for skill loading errors

### Issue 2: Skill Not Triggering

**Symptoms:**
- Skill does not activate automatically
- Manual activation required
- Agent does not recognize skill relevance

**Resolution:**
1. Review SKILL.md description
2. Check trigger keywords match user input
3. Update skill description if needed
4. Test with explicit skill activation

### Issue 3: Action Failed

**Symptoms:**
- Skill action returns error
- Unexpected behavior
- Incomplete results

**Resolution:**
1. Check action parameters
2. Verify dependencies are available
3. Review skill documentation
4. Test with minimal parameters

## Diagnostic Commands

Check skill installation, view skill definition, and test skill activation.

For issues not covered here, check skill documentation, search GitHub issues, or contact the skill team via Telegram.
