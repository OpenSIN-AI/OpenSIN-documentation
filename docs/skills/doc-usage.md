# Doc — Usage Guide

> **Skill:** doc | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Doc** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about doc.

## Manual Activation

```bash
opencode run "use skill doc to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use doc
```

### Example 2: Advanced Usage
```
/use doc with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/doc/`
2. Verify the SKILL.md exists
3. Restart OpenCode
