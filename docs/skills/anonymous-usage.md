# Anonymous — Usage Guide

> **Skill:** anonymous | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Anonymous** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about anonymous.

## Manual Activation

```bash
opencode run "use skill anonymous to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use anonymous
```

### Example 2: Advanced Usage
```
/use anonymous with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/anonymous/`
2. Verify the SKILL.md exists
3. Restart OpenCode
