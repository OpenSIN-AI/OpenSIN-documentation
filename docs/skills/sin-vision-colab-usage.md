# Sin Vision Colab — Usage Guide

> **Skill:** sin-vision-colab | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Sin Vision Colab** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about sin vision colab.

## Manual Activation

```bash
opencode run "use skill sin-vision-colab to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use sin-vision-colab
```

### Example 2: Advanced Usage
```
/use sin-vision-colab with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/sin-vision-colab/`
2. Verify the SKILL.md exists
3. Restart OpenCode
