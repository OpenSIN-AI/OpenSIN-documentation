# Omoc Plan Swarm — Usage Guide

> **Skill:** omoc-plan-swarm | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Omoc Plan Swarm** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about omoc plan swarm.

## Manual Activation

```bash
opencode run "use skill omoc-plan-swarm to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use omoc-plan-swarm
```

### Example 2: Advanced Usage
```
/use omoc-plan-swarm with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/omoc-plan-swarm/`
2. Verify the SKILL.md exists
3. Restart OpenCode
