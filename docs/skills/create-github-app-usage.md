# Create Github App — Usage Guide

> **Skill:** create-github-app | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Create Github App** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about create github app.

## Manual Activation

```bash
opencode run "use skill create-github-app to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use create-github-app
```

### Example 2: Advanced Usage
```
/use create-github-app with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/create-github-app/`
2. Verify the SKILL.md exists
3. Restart OpenCode
