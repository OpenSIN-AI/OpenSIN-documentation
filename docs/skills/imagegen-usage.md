# Imagegen — Usage Guide

> **Skill:** imagegen | **Category:** OpenCode Skill

## Overview

This guide provides detailed usage instructions for the **Imagegen** skill in OpenCode.

## When to Use

The skill is automatically triggered when the user asks about imagegen.

## Manual Activation

```bash
opencode run "use skill imagegen to [task]"
```

## Examples

### Example 1: Basic Usage
```
/use imagegen
```

### Example 2: Advanced Usage
```
/use imagegen with advanced options
```

## Configuration

No additional configuration required.

## Troubleshooting

If the skill doesn't trigger:
1. Check if the skill is installed: `ls ~/.config/opencode/skills/imagegen/`
2. Verify the SKILL.md exists
3. Restart OpenCode
