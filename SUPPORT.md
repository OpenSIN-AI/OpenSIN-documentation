# Support

## Where to get help

- Start with the [README](README.md) and the docs site at https://docs.opensin.ai
- Use [GitHub Issues](https://github.com/OpenSIN-AI/OpenSIN-documentation/issues) for docs bugs or broken links
- Use [GitHub Discussions](https://github.com/OpenSIN-AI/OpenSIN-documentation/discussions) if community discussion is enabled
- Use [SECURITY.md](SECURITY.md) for private reporting paths

## What to include

- The exact page, URL, or file path
- What you expected to see
- What actually happened
- Evidence: screenshot, log, or repro steps

## What not to use support for

- Secrets, tokens, or private screenshots
- Runtime/product ownership questions that belong in another repo
- Unverified public-surface claims
